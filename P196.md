给定的 Laplace 方程
\[
\frac{\partial^2 \xi}{\partial x^2} + \frac{\partial^2 \xi}{\partial y^2} = 0 \]
\[
\frac{\partial^2 \eta}{\partial x^2} + \frac{\partial^2 \eta}{\partial y^2} = 0 
\]
其中 $\xi$ 和 $\eta$ 是计算坐标（因变量），$x$ 和 $y$ 是物理坐标（自变量）。
现在需要反转角色，使 $x$ 和 $y$ 成为因变量，而 $\xi$ 和 $\eta$ 成为自变量。将原 Laplace 方程（在物理平面中对 $\xi,\eta$）转换为计算平面中对 $x,y$ 的方程。
\[
\alpha \frac{\partial^2 x}{\partial \xi^2} - 2\beta \frac{\partial^2 x}{\partial \xi \partial \eta} + \gamma \frac{\partial^2 x}{\partial \eta^2} = 0 \\
\alpha \frac{\partial^2 y}{\partial \xi^2} - 2\beta \frac{\partial^2 y}{\partial \xi \partial \eta} + \gamma \frac{\partial^2 y}{\partial \eta^2} = 0 \\
\]
其中：
\[
\alpha = \left( \frac{\partial x}{\partial \eta} \right)^2 + \left( \frac{\partial y}{\partial \eta} \right)^2
\]
\[
\beta = \frac{\partial x}{\partial \xi} \frac{\partial x}{\partial \eta} + \frac{\partial y}{\partial \xi} \frac{\partial y}{\partial \eta}
\]
\[
\gamma = \left( \frac{\partial x}{\partial \xi} \right)^2 + \left( \frac{\partial y}{\partial \xi} \right)^2
\]
## 推导步骤：
1. **一阶偏导数的变换关系**：
    已知：$
    x = x(\xi, \eta), \quad y = y(\xi, \eta)
    $，其雅可比矩阵为：$
    \begin{bmatrix}
    x_\xi & x_\eta \\
    y_\xi & y_\eta
    \end{bmatrix}
    $，其中 $x_\xi = \frac{\partial x}{\partial \xi}$，其余符号定义类似。

    雅可比行列式为：
    \[
    J = x_\xi y_\eta - x_\eta y_\xi
    \]

    根据逆函数定理，正变换的导数由逆矩阵给出：
    \[
    \begin{bmatrix}
    \xi_x & \xi_y \\
    \eta_x & \eta_y
    \end{bmatrix}
    = J^{-1} = \frac{1}{J} \begin{bmatrix}
    y_\eta & -x_\eta \\
    -y_\xi & x_\xi
    \end{bmatrix}
    \]
    因此，得到一阶导数关系式：
    \[
    \xi_x = \frac{y_\eta}{J}, \quad \xi_y = -\frac{x_\eta}{J} 
    \eta_x = -\frac{y_\xi}{J}, \quad \eta_y = \frac{x_\xi}{J} 
    \]

3. **二阶偏导数 $\frac{\partial^2 \xi}{\partial x^2}$ 的变换**：
我们计算 $\frac{\partial}{\partial x} \left( \xi_x \right)$。首先，将 $\xi_x$ 视为 $\xi$ 和 $\eta$ 的函数，应用链式法则：
\[
\frac{\partial}{\partial x} (\xi_x) = \xi_x \frac{\partial (\xi_x)}{\partial \xi} + \eta_x \frac{\partial (\xi_x)}{\partial \eta}
\]
代入 $\xi_x = \frac{y_\eta}{J}$ 和 $\eta_x = -\frac{y_\xi}{J}$：
\[
\begin{aligned}
    \frac{\partial^2 \xi}{\partial x^2} &= \left( \frac{y_\eta}{J} \right) \frac{\partial}{\partial \xi} \left( \frac{y_\eta}{J} \right) + \left( -\frac{y_\xi}{J} \right) \frac{\partial}{\partial \eta} \left( \frac{y_\eta}{J} \right)
\\
&= \frac{y_\eta}{J} \cdot \frac{\partial}{\partial \xi} \left( \frac{y_\eta}{J} \right) - \frac{y_\xi}{J} \cdot \frac{\partial}{\partial \eta} \left( \frac{y_\eta}{J} \right) \tag{1}
\end{aligned}\]
计算括号内的偏导数：
\[
\frac{\partial}{\partial \xi} \left( \frac{y_\eta}{J} \right) = \frac{1}{J} \frac{\partial y_\eta}{\partial \xi} - \frac{y_\eta}{J^2} \frac{\partial J}{\partial \xi}
= \frac{1}{J} y_{\eta\xi} - \frac{y_\eta}{J^2} \frac{\partial J}{\partial \xi}
\\
\frac{\partial}{\partial \eta} \left( \frac{y_\eta}{J} \right) = \frac{1}{J} \frac{\partial y_\eta}{\partial \eta} - \frac{y_\eta}{J^2} \frac{\partial J}{\partial \eta}
= \frac{1}{J} y_{\eta\eta} - \frac{y_\eta}{J^2} \frac{\partial J}{\partial \eta}
\]
将这两个结果代入式 (1)：
\[
\frac{\partial^2 \xi}{\partial x^2} = \frac{y_\eta}{J} \left( \frac{1}{J} y_{\eta\xi} - \frac{y_\eta}{J^2} \frac{\partial J}{\partial \xi} \right) - \frac{y_\xi}{J} \left( \frac{1}{J} y_{\eta\eta} - \frac{y_\eta}{J^2} \frac{\partial J}{\partial \eta} \right)
\\
= \frac{y_\eta y_{\eta\xi}}{J^2} - \frac{y_\eta^2}{J^3} \frac{\partial J}{\partial \xi} - \frac{y_\xi y_{\eta\eta}}{J^2} + \frac{y_\xi y_\eta}{J^3} \frac{\partial J}{\partial \eta} 
\]

5. **将 $\frac{\partial^2 \xi}{\partial x^2} + \frac{\partial^2 \xi}{\partial y^2}$ 用 $x,y$ 对 $\xi,\eta$ 的导数表示**：
   通过链式法则和上述逆矩阵，可以导出：
   \[
   \frac{\partial^2 \xi}{\partial x^2} + \frac{\partial^2 \xi}{\partial y^2} = \frac{1}{J^2} \left[ \alpha \frac{\partial^2 x}{\partial \xi^2} - 2\beta \frac{\partial^2 x}{\partial \xi \partial \eta} + \gamma \frac{\partial^2 x}{\partial \eta^2} \right] = 0
   \]
   类似地，对 $\eta$ 有：
   \[
   \frac{\partial^2 \eta}{\partial x^2} + \frac{\partial^2 \eta}{\partial y^2} = \frac{1}{J^2} \left[ \alpha \frac{\partial^2 y}{\partial \xi^2} - 2\beta \frac{\partial^2 y}{\partial \xi \partial \eta} + \gamma \frac{\partial^2 y}{\partial \eta^2} \right] = 0
   \]
   其中：
   \[
   \alpha = \left( \frac{\partial x}{\partial \eta} \right)^2 + \left( \frac{\partial y}{\partial \eta} \right)^2
   \]
   \[
   \beta = \frac{\partial x}{\partial \xi} \frac{\partial x}{\partial \eta} + \frac{\partial y}{\partial \xi} \frac{\partial y}{\partial \eta}
   \]
   \[
   \gamma = \left( \frac{\partial x}{\partial \xi} \right)^2 + \left( \frac{\partial y}{\partial \xi} \right)^2
   \]

1. **得到最终方程**：

   由于 $J^2 \neq 0$，所以：
   \[
   \alpha \frac{\partial^2 x}{\partial \xi^2} - 2\beta \frac{\partial^2 x}{\partial \xi \partial \eta} + \gamma \frac{\partial^2 x}{\partial \eta^2} = 0 \]
   \[
   \alpha \frac{\partial^2 y}{\partial \xi^2} - 2\beta \frac{\partial^2 y}{\partial \xi \partial \eta} + \gamma \frac{\partial^2 y}{\partial \eta^2} = 0 
   \]


因此，通过上述推导，得到了边界拟合网格生成中使用的椭圆型方程 (5.69) 和 (5.70)。
